# npm
## npm包维护升级

npm-check检查更新

```bash
npm install -g npm-check
npm-check
```

npm-upgrade更新

```bash
npm install -g npm-upgrade
npm-upgrade
```

## 基本操作

npm init 初始化一个项目

```bash
npm init react-app ./my-react-app
npm init egg --type=simple
```
